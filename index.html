Git vs GitHub

Git

A command-line tool on your laptop. Tracks file changes. Works offline. Free, open-source. Created by Linus Torvalds in 2005.

GitHub

A webside that host gif repos and adds collaboration on top. Pull requests, issues, code review, CI, the social layer.

Daily loop

  • Edit
  • Stage
  • Commit
  • Push

Command Reference

  • git init:This creates a .git directory that tracks your project. It's now a local Git repo.
  • git status:Shows what files are untracked (not staged) or modified. Helpful before committing.
  • git add:Stages the file so it will be included in the next commit.
  • git commit:Commits all staged files to your local repo with a message.
  • git push:Sends your local commits to the remote repository. Often used after commit.
  • git pull:Downloads and merges new changes from the remote branch.
  • git branch:Branches help isolate features or fixes. Always create a branch for new work!
  • git merge:Combines the changes from the feature branch into your main branch.
